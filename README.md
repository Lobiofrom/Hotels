Тестовое задание одной из компаний - приложение, имитирующее бронь гостиницы. Состоит из четырех экранов. 
Использованы такие технологии, как: 
- Многомодульность,
- Jetpack Compose,
- Compose Navigation,
- Clean Architecture,
- Kotlin,
- Корутины,
- Flow,
- LiveData,
- Dagger,
- MVVM.
